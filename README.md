# End-to-End Machine Learning Project

## Project Overview
This project demonstrates an end-to-end machine learning workflow.

## Tech Stack
- Python 3.8
- Pandas, NumPy
- Scikit-learn
- Streamlit

## Steps
1. Data Collection
2. Data Preprocessing
3. Model Training
4. Model Evaluation
5. Deployment

## How to Run
conda activate venv
pip install -r requirements.txt
streamlit run app.py
